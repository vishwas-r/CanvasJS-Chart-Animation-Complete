CanvasJS Chart Animation Complete Event

This plugin allows you to bind animationComplete event handler in CanvasJS Chart. Animation Complete Event gets fired after the dataseries has finished its initial animation. Check out Github Repository for more info / examples.

Live Example

How to Use?

var chart = new CanvasJS.Chart("chartContainer", {
	.
	.
	.
	//Chart Options
	animationEnabled: true,
	animationComplete: function(e) {
		//Things to do once chart animation is complete
	}
	.
	.
	.
});
chart.render();
